It's like having a writer for your save file.

Every command reads what's actually happening in your game — who lives in the household, their traits, their relationships, their mood, where they are — and sends it to Claude as context. The AI doesn't make up sims that don't exist or invent jobs they don't have. It writes for the people in front of it.

A flirty teen texts in lowercase with emoji. An elder writes long, warm paragraphs. A grandparent dotes; an enemy sneers. A Goofball cracks jokes mid-call. The voice fits.

What it does

Phone Calls

Random sims call you in character — your kid's friend, your boss, your ex. Each voice shaped by traits, age, mood, and your shared history.

Text Messages

Quick check-ins, group drama, gossip. Hit Reply on the popup and they keep the conversation going — full back-and-forth, in character.

Story Updates

Soap-opera narration of what's happening in your household, written like a let's-play. Continues from past entries so it builds across sessions.

Random Events

Surprises that fit your sims' actual lives — promotions, fights, weird neighbours, family secrets. Always actionable; tells you exactly what to do next.

Storylines

Multi-session 3-act arcs with concrete gameplay goals — raise a skill, hit a career level, complete an aspiration. Pick a theme or let Claude pick one.

Auto-Events

Sits quietly in the background and fires content while you play. Set the interval, set the mix (calls, texts, stories, events), forget about it.

Commands at a glance

Open the cheat console with Ctrl+Shift+C, type a command, press Enter.

Phone

  • claude.call — incoming call
  • claude.text — incoming text
  • claude.sendtext Bella Goth hey!
  • claude.sendcall Bella Goth I have news
  • claude.reply <message>

Storytelling

  • claude.story — narrative update
  • claude.storyline — 3-act arc
  • claude.storyline_theme romance
  • claude.drama — household drama
  • claude.dialogue — in-character lines

Events & goals

  • claude.event — random event
  • claude.goals — session goals
  • claude.challenge — gameplay challenge
  • claude.auto_events on

System

  • claude.status — setup & all commands
  • claude.chat <message> — freeform
  • claude.journal — recent history
  • claude.reload — after editing config

Get started

  1. 1

    Download the mod

    Grab the latest release from the downloads page. You need two files: ClaudeAI.ts4script and claude_config.cfg.

  2. 2

    Get your Claude API key

    The mod uses Claude (made by Anthropic) to write the dialogue and stories. You need your own API key — it's free to sign up, and you only pay for what you use. A whole evening of play is usually 25–50 cents.

    Step-by-step: getting your API key

    1. Go to console.anthropic.com and click Sign up. Use Google or your email — both work fine.
    2. Verify your email and phone number if Anthropic asks. This is a one-time anti-spam check.
    3. Add a credit card and buy a little credit. Anthropic needs a card on file before they'll let you use the API. Go to Settings → Billing and add $5 (the minimum). That's enough for thousands of in-game calls and texts — most players never come close to using it up in a month.
    4. Create the key. In the left sidebar click API Keys (or go to console.anthropic.com/settings/keys), then click the Create Key button.
    5. Name it something memorable like "Sims 4 mod" and click Create.
    6. Copy the key immediately and keep it private. It looks like sk-ant-api03-… followed by a long string. You only see it once — if you close the dialog without copying, you'll need to make a new one.
      Treat this key like a password. Anyone who has it can spend money on your Anthropic account.
      • Save it in a password manager (1Password, Bitwarden, your browser's saved passwords) — not in a plain text file on your desktop.
      • Never paste it in Discord, Reddit, GitHub, screenshots, or any chat — even private DMs aren't safe long-term.
      • If it ever leaks, go to the API keys page, revoke the old key, and make a new one.

    That's it for the Anthropic side — the key is yours. Keep it safe.

  3. 3

    Drop the mod files in your Mods folder

    Move both downloaded files into:

    Documents/Electronic Arts/The Sims 4/Mods/

    If you have Sims 4 mods already, you've been here before — same folder.

  4. 4

    Paste your API key into the config

    Open claude_config.cfg with any text editor (Notepad, VS Code, TextEdit on Mac). Find this line near the top:

    api_key = YOUR_API_KEY_HERE

    Replace YOUR_API_KEY_HERE with the key you copied. It should look like:

    api_key = sk-ant-api03-Xy7…(your full key)…

    Save the file.

  5. 5

    Enable script mods & launch the game

    In The Sims 4, open Game Options → Other and tick both:

    • Enable Custom Content and Mods
    • Script Mods Allowed

    Restart the game. When the main menu loads, you'll see a notification popup confirming the mod is active.

    Inside any household, open the cheat console with Ctrl+Shift+C and type claude.status to see all the commands and confirm everything's wired up.
